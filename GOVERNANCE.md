# Governance

Product Factory OS is maintained by hihol-labs.

## Decision Process

Major changes should be documented through issues, pull requests, or ADRs.

Examples of major changes:

- Runtime state format
- Starter pack contract
- Skill contract
- Validation behavior
- Commercial boundary
- License change

## Open Core Boundary

The open source repository contains the community runtime.

Commercial extensions may be developed in separate private repositories and referenced from this repository as roadmap or product documentation.
