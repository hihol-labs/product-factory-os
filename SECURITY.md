# Security Policy

## Reporting

Please report security issues privately to the maintainers.

Do not publish exploitable vulnerabilities or secrets in public issues.

## Scope

Security reports may cover:

- PFO runtime scripts
- Generated starter packs
- Validators
- Integration exporters
- Packaging scripts

Generated products belong to their authors. Security issues in a generated product should be reported to that product owner unless the issue is caused by PFO-generated insecure defaults.
