# Pricing Model

This is a suggested pricing structure.

## Community

Price: Free

Includes:

- Open source runtime
- Basic starter packs
- Local validators
- Local dashboard shell
- Local state and memory

## Pro

Price: paid monthly or annual license

Includes:

- Premium starter packs
- Advanced generated-project validators
- Pro dashboard features
- Advanced benchmarks
- Priority updates

## Team

Price: per-seat or per-workspace

Includes:

- Shared workspace
- Team dashboard
- Approval workflows
- GitHub/Linear/Notion sync
- Shared template registry
- Organization memory

## Enterprise

Price: custom

Includes:

- Private deployment
- SSO/RBAC
- Audit logs
- Policy enforcement
- Custom packs
- Support SLA
- Onboarding
