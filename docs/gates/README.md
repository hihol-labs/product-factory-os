# Universal Runtime Gates

These gates are product-agnostic. Product Factory OS owns the mechanism; each project owns its concrete contracts in `.pfo/`.

Required universal gates:

- Scope Lock
- Data Authenticity
- Golden Flow Tests
- Regression Contract
- Fallback Policy
- Diff Risk Classifier
- Security Review Evidence
- No Silent Substitution
- Execution Policy
- Permission Matrix
- Verification Contract
- Tool Capability Registry
- Learning Promotion

Optional product-specific gates:

- SEO Growth Guarantee
