# Commercial Strategy

Product Factory OS can be monetized without closing the core.

## Commercial Products

### PFO Pro

Paid local/professional layer:

- Premium starter packs
- Advanced validators
- Advanced benchmark packs
- Private template registry
- Pro dashboard features

### PFO Cloud

Hosted control plane:

- Project dashboard
- Gate status
- Execution graph visualization
- Team activity
- Metrics
- Repo sync
- Approval workflows

### PFO Enterprise

Enterprise deployment and support:

- Private deployment
- SSO/RBAC
- Audit logs
- Policy enforcement
- Custom starters
- Team onboarding
- Support SLA

### Services

Implementation and enablement:

- Workspace setup
- Agency/startup-studio onboarding
- Custom product factory design
- Premium pack creation
- Team training

## Commercial Boundary

The open source repository may describe commercial extensions, but should not include closed commercial source code.

Recommended private/commercial repositories:

```text
pfo-premium-packs
pfo-cloud
pfo-enterprise
pfo-customer-templates
```
