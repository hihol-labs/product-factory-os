# Starter Pack Strategy

Starter packs are the clearest monetization path after the open source runtime.

## Community Packs

Included in open source:

- Basic SaaS
- Basic bot
- Basic API
- Basic landing
- Basic CLI
- Basic scraper
- Basic e-commerce
- Basic mini app
- Basic internal automation

## Premium Pack Ideas

- SaaS with auth, billing, admin, onboarding, and analytics
- Telegram bot with CRM and notifications
- AI agent SaaS
- Marketplace starter
- Multi-tenant B2B SaaS
- Scraper with queues, proxies, deduplication, and monitoring
- Internal tools with RBAC and audit logs
- E-commerce with payment webhooks and order management

## Pack Contract

Every pack should define:

- Product type
- Stack
- Folder structure
- Generated files
- Required docs
- Test matrix
- Security requirements
- Deployment path
- Upgrade notes
