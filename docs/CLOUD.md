# PFO Cloud

PFO Cloud is the hosted control plane for Product Factory OS.

## MVP Scope

- Project list
- Current state
- Current execution node
- Gate status
- Blockers
- PFO report
- Metrics
- GitHub repository connection
- Route snapshot and hook-gate status

## Later Scope

- Team workspaces
- Approval workflows
- Execution graph visualization
- Hosted template registry
- Linear/Notion sync
- Organization memory
- Audit logs
- Managed runners
- Organization-specific hook policies

## Data Model

- Organization
- Workspace
- Project
- State snapshot
- Execution node
- Gate result
- Integration connection
- Template pack
- Route snapshot
- Hook gate result
